ORC是LOR的本地改动版，可以通简单的命令快速的生成Openresty的配置文件和目录结构：


安装[ORC](https://coding.net/u/shengyang/p/orc/git)，然后执行下下面的操作：

1.安装

sudo sh install.sh


2.生成工程

orc project testcase


3.控制工程

进入testcase目录

orc start dev

orc stop dev

orc reload dev
