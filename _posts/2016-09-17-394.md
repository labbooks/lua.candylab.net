Moonscript如何显示复选框

```lua
  "/checkbok": =>
      @url_for: =>
      @html -> 
        form method: "POST", action: @url_for(), ->
          input type: "hidden",
            name: "csrf_token", value: @csrf_token
          input type: "checkbox", name: "username"
          input type: "checkbox", name: "password"
```
