昨天安装老的版本的Django,删除Python，误删除了Unity．
结果发现桌面的左边栏和Terminal的标题栏都没了．

恢复安装一下就好了．

```
sudo apt-get install unity
sudo apt-get install gnome-terminal
```
这样重启系统就恢复了．
