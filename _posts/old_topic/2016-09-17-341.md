env的实现，就是下面一小段的C语言实现。

```c
#include <stdio.h>  
extern char**environ;  
int main ()  
{  
   char**var;  
   char *str;  
   for (var =environ;*var !=NULL;++var)  
       printf ("%s\n",*var); 
   return 0;  
}
```
