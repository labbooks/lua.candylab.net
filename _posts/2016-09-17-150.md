Lua有些符号在用户字符替换的时候，可能会发生转义。
今天群里的一位小伙伴，就需要到类似的问题，比如说要将字符串中的'（'变成'['。

解答问题的伙伴给出如下代码：

```lua
a = '123.(456)'
c = string.gsub(a, '%(', '[')
print(c)
```

输出的结果是：
```
123.[456)
```



ngx.re.gsub是如何使用转义的，下面给一个实际的正则的例子：
```
sleep\((\s*)(\d*)(\s*)\)
```

主要是"%"符号，起到了告诉系统转义的用途。
